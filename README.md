# -
오픈소스 과제
수학 공식 정리 웹페이지 (GitHub Pages 프로젝트)

본 프로젝트는 오픈소스SW활용 수업 과제 및 경진대회 제출을 위해 제작한 수학 공식 정리 웹페이지입니다.  
고등수학부터 대학미적분까지의 핵심 공식을 한 페이지에서 정리하여 누구나 쉽게 참고할 수 있도록 구성했습니다.

웹사이트는 GitHub Pages를 통해 배포되며, HTML/CSS와 MathJax 등 오픈소스 라이브러리를 사용해 제작되었습니다.

## 주요 기능 및 구성

### 수학 공식 정리 페이지
- 기본 공식(일차식/이차식/항등식)
- 기초 미적분 공식
- 대학미적분 핵심 공식 (극한/연속/미분/적분/다변수/테일러 전개 등)
- 삼각함수 기본 항등식/덧셈정리/배각공식
- 확률·통계의 기초(조합/기댓값/분산)

### MathJax 적용
LaTeX 문법을 활용해 수학식을 아름답고 정확하게 렌더링.

### 반응형 디자인
모바일/태블릿/PC 화면 크기에 자동으로 대응.

## 사용한 기술 스택

- HTML / CSS
- MathJax — 수학식 렌더링 오픈소스 라이브러리  
  CDN 사용: https://cdn.jsdelivr.net/npm/mathjax  
- GitHub Pages — 정적 웹사이트 배포
- GitHub Issues / Branch / Pull Request — 버전관리 및 협업 흐름 적용

## 🧩 GitHub 협업 방식

본 프로젝트는 GitHub의 다양한 기능을 활용해 진행되었습니다.

1. Issue 생성
   - 기능 개발 계획, 레이아웃 구성, 오류 수정 등을 Issue로 분류

2. Branch 분리 작업 
   - `feature/layout`, `feature/formula`,  
   - `feature/readme` 등의 브랜치를 생성해 병렬 개발

3. Pull Request 및 코드 리뷰 
   - 각 브랜치에서 PR 생성 후 팀원 승인 → main 브랜치로 merge

4. Commit 기록 관리
   - 작은 단위로 기능을 추가하며 의미 있는 커밋 메시지 작성
